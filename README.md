
# VLL Website

Official website for Vardi, Lavie-Loboda & Co.

A bilingual (Hebrew / English) CMS-based website for a boutique law firm, built with Django and Wagtail.

## Setup

See PROJECT_SETUP.md

### Stack

- Python 3.14.4
- Django 6.0.4
- Wagtail 7.3.1
- PostgreSQL (psycopg)
- HTML / CSS
- Render

## Features (Planned)

- Bilingual content (Hebrew RTL + English)
- Structured CMS (no page builder)
- Practice Areas management
- Attorney profiles
- Contact form
- SEO fields per page
- Articles infrastructure

## Development Approach

This project follows a strict workflow:

1. Define SPEC
2. Break into small tasks
3. Execute using Cursor (minimal prompts)
4. Review → test → commit

Cursor is used only for execution, not planning.
