
# Project Setup

## Python Version
3.14.4

## Setup

Create virtual environment:

python3 -m venv .venv

Activate:

source .venv/bin/activate

Install dependencies:

pip install -r requirements.txt

## Dependencies

Django
wagtail
psycopg[binary]
